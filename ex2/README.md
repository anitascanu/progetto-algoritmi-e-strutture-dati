# COMPILAZIONE

- posizionarsi in ../ex2
- creare le cartelle ex2/bin e ex2/build se non sono presenti
- make (nella directory corrente ci deve essere il Makefile opportuno)

---

# CANCELLAZIONE

- posizionarsi in ../ex2
- make clean (nella directory corrente ci deve essere il Makefile opportuno)

---

# ESECUZIONE

## PER ESEGUIRE skiplist_main

- compilare
- posizionarsi in ../ex2/bin
- ./skiplist_main `dictionary_file` `correctme_file`

`dictionary_file`: percorso file con le parole da inserire nella skiplist

`correctme_file`: percorso file con il testo da correggere sulla base del dizionario

## PER ESEGUIRE skiplist_tests

- compilare
- posizionarsi in ../ex2/bin
- ./skiplist_tests
